# Locales

Единый каталог пользовательских текстов для разных поверхностей:
- `bot`
- `web`
- в будущем `admin`, если понадобится

Принцип:
- `common.json` содержит общие действия и базовые строки
- `bot.json` содержит тексты и шаблоны бота
- `web.json` содержит тексты frontend-клиента

Формат:
- JSON
- именованные плейсхолдеры вида `{name}`
- fallback locale сейчас только `ru`

Причина выбора JSON:
- одинаково читается из Python и TypeScript
- не требует отдельного runtime-парсера
- хорошо работает в `vite` и в backend/bot code
