# Архитектура

## Слои
- **API** — единый источник бизнес‑логики, платежей, интеграций и доступа к данным.
- **Bot** — канал продаж и поддержки, тонкий клиент.
- **WebApp** — витрина и управление подпиской, тонкий клиент.

## Модули
- Пользователи
- Подписки
- Платежи
- Провижининг (Remnawave)

## Потоки
- Бот/Вебапп → API → Remnawave
- Платежный провайдер → API (webhook) → Remnawave → уведомление в бот
