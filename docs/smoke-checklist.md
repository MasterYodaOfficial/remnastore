# Smoke Checklist

Этот чеклист выполняется перед первым merge релиз-кандидата в `main` и перед production deploy.

## Browser Web

- [ ] Открывается `web`, приложение не падает на boot
- [ ] Работает browser auth
- [ ] После входа грузятся баланс, подписка и основные панели
- [ ] Открывается каталог тарифов
- [ ] Создается или возобновляется pending top-up flow
- [ ] Работает просмотр истории баланса
- [ ] Работает просмотр уведомлений
- [ ] Работает реферальная страница и копирование/шаринг ссылки
- [ ] Открывается FAQ и legal pages

## Telegram Mini App

- [ ] Mini App открывается из бота без white screen
- [ ] Telegram auth/session корректно подхватываются
- [ ] Видны баланс, подписка, тарифы и настройки
- [ ] Работает переход к оплате и возврат в живой state
- [ ] Не протекают сырые backend/internal ошибки в UI

## Bot

- [ ] `/start` работает
- [ ] Главное меню открывается без ошибок
- [ ] Кнопка открытия Mini App работает
- [ ] Deep-link/referral flow не сломан
- [ ] Критичные системные сообщения и уведомления доходят до пользователя

## Admin

- [ ] Логин в admin проходит
- [ ] Поиск аккаунта работает
- [ ] Открывается карточка аккаунта
- [ ] Видны ledger, payments, withdrawals, referrals и events
- [ ] Доступны безопасные admin actions на тестовом аккаунте

## Observability

- [ ] В логах нет утечки токенов, `initData`, payment payload secrets
- [ ] Нет всплеска `5xx`, auth ошибок и failed webhook delivery
- [ ] Нет застрявших критичных background jobs

## Результат

- [ ] Smoke пройден полностью
- [ ] Отклонения задокументированы и либо устранены, либо явно признаны допустимыми для релиза
