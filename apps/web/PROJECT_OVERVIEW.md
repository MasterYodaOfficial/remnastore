# Обзор проекта VPN Subscription Service

## 📱 Описание

Полнофункциональное React приложение для продажи VPN подписок с поддержкой Telegram WebApp и браузерной авторизации.

## 🎯 Ключевые возможности

### Для пользователей:
- ✅ Управление профилем и балансом
- ✅ Пробный период 7 дней
- ✅ Различные тарифные планы (1 месяц, 3 месяца, 12 месяцев)
- ✅ Реферальная программа (20% с покупок рефералов)
- ✅ Вывод реферальных средств на баланс
- ✅ Адаптивный дизайн в стиле Telegram
- ✅ Темная и светлая тема

### Технические:
- ✅ Telegram WebApp с автоматической авторизацией
- ✅ OAuth авторизация (Google, Yandex, VK) для браузера
- ✅ Backend на Supabase Edge Functions
- ✅ Безопасное хранение данных в KV Store
- ✅ Real-time обновления
- ✅ Toast уведомления

## 🔧 Технологический стек

### Frontend:
- **React 18** - UI библиотека
- **TypeScript** - Типизация
- **Tailwind CSS** - Стилизация
- **Radix UI** - UI компоненты
- **Lucide Icons** - Иконки
- **Sonner** - Toast уведомления
- **Telegram WebApp SDK** - Интеграция с Telegram

### Backend:
- **Supabase Auth** - Аутентификация
- **Supabase Edge Functions** - Serverless API
- **Hono** - Web фреймворк (Deno)
- **KV Store** - База данных

## 🚀 API Endpoints

### Авторизация
- `POST /signup` - Регистрация
- `GET /profile` - Профиль пользователя

### Баланс
- `POST /balance/add` - Пополнение баланса

### Подписки
- `GET /subscription` - Информация о подписке
- `POST /subscription/trial` - Активация пробного периода
- `GET /plans` - Список тарифов
- `POST /plans/buy` - Покупка тарифа

### Рефералы
- `GET /referrals` - Список рефералов
- `POST /referrals/withdraw` - Вывод средств

## 💾 База данных (KV Store)

### Типы данных:
- `user:{userId}` - Профиль пользователя
- `subscription:{userId}` - Подписка
- `plan:{planId}` - Тарифный план
- `referral:{referrerId}:{referralId}` - Реферал
