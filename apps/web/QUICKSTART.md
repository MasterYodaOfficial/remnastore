# 🚀 Быстрый старт

## Что нужно сделать для запуска приложения

### 1. Настроить OAuth провайдеры в Supabase ⚠️

Это **обязательный шаг** для работы авторизации в браузере!

Откройте файл `OAUTH_SETUP.md` и следуйте инструкциям для настройки хотя бы одного провайдера:
- Google (рекомендуется)
- Yandex
- VK

**Важно:** Без настройки OAuth провайдеров авторизация в браузере работать не будет! Только Telegram WebApp будет работать.

### 2. Протестировать в браузере

После настройки OAuth:
1. Откройте приложение в браузере
2. Нажмите на кнопку входа через Google/Yandex/VK
3. Авторизуйтесь
4. Попробуйте:
   - Пополнить баланс (кнопка "Пополнить" в шапке)
   - Активировать пробный период
   - Купить подписку
   - Скопировать реферальную ссылку

### 3. Настроить Telegram WebApp (опционально)

Для работы в Telegram:
1. Создайте бота через [@BotFather](https://t.me/botfather)
2. Используйте команду `/newapp` чтобы создать Web App
3. Укажите URL вашего приложения
4. Настройте кнопку для открытия Web App в боте
5. Откройте приложение через бота

## Демо-функции

Приложение работает в демо-режиме для тестирования:
- ✅ Пополнение баланса работает мгновенно (нажмите "Пополнить")
- ✅ Все тарифы можно купить после пополнения баланса
- ✅ Реферальная ссылка копируется в буфер обмена
- ✅ Можно вывести реферальные средства на баланс

## Важные файлы

- `/README.md` - Полная документация
- `/OAUTH_SETUP.md` - Инструкции по настройке OAuth
- `/src/app/App.tsx` - Главный компонент приложения
- `/supabase/functions/server/index.tsx` - Backend API

## Нужна помощь?

Если что-то не работает:
1. Проверьте настройку OAuth провайдеров в Supabase Dashboard
2. Откройте консоль браузера (F12) для просмотра ошибок
3. Убедитесь, что Supabase проект правильно настроен
