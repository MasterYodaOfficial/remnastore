# VPN Subscription Service

React приложение для продажи VPN подписок с поддержкой Telegram WebApp и браузерной авторизации через OAuth провайдеры (Google, Yandex, VK).

## Возможности

### Основные функции
- ✅ Профиль пользователя с балансом
- ✅ Пополнение баланса (демо-режим)
- ✅ Управление подписками
- ✅ Пробный период (7 дней)
- ✅ Различные тарифные планы
- ✅ Реферальная программа (20% от покупок рефералов)
- ✅ Вывод реферальных средств на баланс
- ✅ Адаптивный дизайн в стиле Telegram
- ✅ Поддержка светлой и темной темы

### Авторизация
- **Telegram WebApp**: Автоматическая авторизация через Telegram ID при открытии в Telegram
- **Браузер**: OAuth авторизация через Google, Yandex, VK

## Структура приложения

### Страницы
1. **Главная** - Профиль, баланс, подписка, реферальная программа
2. **Тарифы** - Список доступных тарифных планов с возможностью покупки
3. **Рефералы** - Детальная информация о рефералах и заработке
4. **Настройки** - Смена темы, языка, выход из аккаунта

### Компоненты
- `Header` - Шапка с профилем и балансом
- `BottomNav` - Нижняя навигация
- `SubscriptionCard` - Карточка подписки с прогресс-баром
- `ReferralCard` - Карточка реферальной программы
- `PlansPage` - Страница с тарифами
- `TopUpModal` - Модальное окно пополнения баланса

## Backend API

Сервер на Supabase Edge Functions (Deno + Hono) предоставляет следующие эндпоинты:

### Авторизация
- `POST /signup` - Регистрация нового пользователя
- `GET /profile` - Получение профиля пользователя

### Баланс
- `POST /balance/add` - Пополнение баланса

### Подписки
- `GET /subscription` - Получение информации о подписке
- `POST /subscription/trial` - Активация пробного периода
- `GET /plans` - Получение списка тарифов
- `POST /plans/buy` - Покупка тарифного плана

### Рефералы
- `GET /referrals` - Получение списка рефералов
- `POST /referrals/withdraw` - Вывод реферальных средств

## Настройка

### 1. OAuth провайдеры

Для работы авторизации в браузере необходимо настроить OAuth провайдеры в Supabase. Подробные инструкции находятся в файле `OAUTH_SETUP.md`.

**Важно:** Обязательно настройте хотя бы один OAuth провайдер (Google, Yandex или VK), иначе браузерная авторизация работать не будет.

### 2. Telegram WebApp

Для работы в Telegram:
1. Создайте бота через [@BotFather](https://t.me/botfather)
2. Настройте Web App URL вашего приложения
3. Откройте приложение через бота в Telegram

## Использование

### В браузере
1. Откройте приложение в браузере
2. Выберите OAuth провайдер (Google, Yandex, VK)
3. Авторизуйтесь
4. Пополните баланс и купите подписку

### В Telegram
1. Откройте приложение через Telegram бота
2. Авторизация произойдет автоматически
3. Используйте все функции приложения

## Демо-режим

Приложение работает в демо-режиме:
- Пополнение баланса происходит мгновенно без реальной оплаты
- Все транзакции сохраняются в базе данных Supabase
- Реферальная система полностью функциональна

## Технологии

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Supabase (Auth, Edge Functions, KV Store)
- **UI Components**: Radix UI, Lucide Icons
- **Telegram**: Telegram WebApp SDK
- **Notifications**: Sonner (Toast)
